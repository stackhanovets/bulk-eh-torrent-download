# bulk-hath-torrent-download

Get SadPanda torrent files and download them using qBittorrent Web UI API.

## Requirements

- Disable browser file save prompt;
- Install qBittorrent, enable the Web UI and set up credentials;

## Run

1. Log in into SadPanda;
2. Run the JS code in your browser console;
3. Add the qBittorrent credentials into the Bash code;
3. Run the Bash code in your shell / Git Bash;
